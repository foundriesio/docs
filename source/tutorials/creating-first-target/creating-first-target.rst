Creating Your First Target
==========================

In the tutorial :ref:`tutorial-gs-with-docker`, you created an application locally.
This provided an introduction to Docker containers and ``docker-compose`` apps.
In this tutorial, you will learn what a **Target** is, how to generate one, and how your device *consumes* it.

.. note::

  Estimated Time to Complete this Tutorial: 15 minutes

Prerequisites
-------------

- Completed the :ref:`tutorial-gs-with-docker` tutorial.
- Completed the :ref:`gs-install-fioctl` section.
- Completed the :ref:`gs-flash-device` section.
- Completed the :ref:`gs-register` section.
